# Rayluno Delivery Roadmap

## Milestone 1 — Personal tasks

Persistent bilingual tasks with priorities and due dates.

## Milestone 2 — Daily agenda and reminders

Due-time reminders, overdue detection, snooze, completion, and a Today summary.

## Milestone 3 — Verified skills

Permission manifests, confirmation gates, and execution receipts for consequential actions.

## Milestone 4 — Personal memory

Explicit remember/forget controls, provenance, and local retrieval.

## Milestone 5 — Multi-step workflows

Bounded plans that compose approved skills and pause at confirmation boundaries.

## Milestone 6 — Judge release

Windows build, polished command center, deterministic demo fixtures, public video, and reproducible verification.
