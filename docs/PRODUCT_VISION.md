# Rayluno Product North Star

Rayluno is a local-first bilingual personal operating layer for Windows. It should understand natural Arabic and English requests, turn them into explicit plans, ask for confirmation when an action changes the system or external data, and produce a clear receipt of what happened.

## Judge moment

A strong demo should prove four things in under three minutes:

1. Rayluno understands a natural voice request.
2. It creates a useful multi-step plan rather than mapping one phrase to one hard-coded action.
3. It protects the user with scoped permissions and visible confirmation.
4. It remembers outcomes locally and can continue the workflow later.

## Product pillars

- **Personal:** tasks, reminders, daily agenda, preferences, and user-approved memory.
- **Capable:** composable skills instead of an unrestricted shell.
- **Private:** local processing and local storage by default.
- **Verifiable:** every consequential action is explainable and auditable.
- **Bilingual:** Arabic and English are first-class paths, not translated afterthoughts.

## Near-term milestones

1. Personal task core.
2. Reminder engine and daily agenda.
3. Permission-gated skills with execution receipts.
4. Local personal memory with explicit remember/forget controls.
5. Multi-step judge workflow and desktop command center.
