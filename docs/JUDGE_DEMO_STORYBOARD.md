# Judge Demo Storyboard

## Scene 1 — Personal context

User says in Arabic:

```text
يا رايلونو، ذكرني بعد عشر دقائق أرسل نسخة العرض للحكام، وخليها أولوية عالية.
```

Rayluno responds with a structured confirmation showing the reminder time, priority, and local-only storage.

## Scene 2 — Daily command center

User asks:

```text
ما خطتي اليوم؟
```

The Today surface groups overdue, due now, later today, and unscheduled tasks. It highlights one recommended focus item without pretending to know more than the stored data.

## Scene 3 — Proactive reminder

When the reminder becomes due, Rayluno presents a single reminder card and speaks it once. The user can complete or snooze it.

## Scene 4 — Safe execution

User asks for a consequential system action. Rayluno shows the selected skill, arguments, permission, risk, and confirmation requirement before execution, then emits an execution receipt.

## Judge takeaway

Rayluno is not a chatbot with a microphone. It is a local personal operating layer that remembers user-approved commitments, manages time, and executes bounded skills with verifiable control.
